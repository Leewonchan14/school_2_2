데이터의 사전

1) 사용자 데이터
실제로 이용하는 데이터를 저장하는 일반 데이터 베이스

2)시스템 데이터


3)메타 데이터
데이터에 대한 데이터(ex 하나 검색하면 관련된 정보가 같이 검색됨)


데이터 독립성

응용프로그램과 데이터 사이의 독립성을 제공

1... 데이터베이스

2... 스키마와 인스턴스
1) 스키마 : 구조체 비슷
2) 인스턴스 : 스키마에 따라 실제로 저장된 값

3... 데이터 베이스 구조(관점에 따라 세가지로 나눔)

1)외부단계
2)개념단계
3)내부단계

4... 데이터의 독립성 :
하위 스키마를 변경하더라도 상위 스키마가 손상되지 않는 특성

1) 논리적데이터 독립성
개념 스키마 변경에도 외부스키마 변경x

2) 물리적 데이터 독립성
내부 스키마 변경에도 개념 스키마 변경x

5... 데이터 사전
시스템 카탈로그 : 걍 데이터 저장장소라 생각하면 편하다

6... 데이터 디렉토리
데이터 사전에 ㅈ장된 데이터의 위치정보를 저장한 시스템
일반사용자는 접근금지

7... 사용자 데이터 베이스
사용자가 실제로 이용하는 데이터가 저장되어있는 시스템

8... 데이터 베이스 사용자,관리자,최종사용자,응용프로그래머

1) 사용자
ㄱ.관리자
ㄴ.응용프로그래머. : 데이터 언어 이용, 프로그램 작성하는사람
ㄷ.최종 사용자 : 삽입삭제수정관리 하는사람

2) 관리자 : 시스템 운영 및 관리 하는 사람
