1. 데이터 베이스의 필요성

ㄱ. 데이터와, 정보의 차이
ㄴ. 데이터베이스의 필요성
ㄷ. 데이터베이스의 정의에 숨겨진 의미와 주요 특징을 이해한다.

ㄱ. 데이터 베이스의 필요성

1. 데이터와 정보의 차이

1) 데이터 : 사실이나. 값
2) 정보 : 데이터를 처리한 결과물
3) 정보 처리 : 데이터에서 정보를 추출하는 과정 또는 방법
	4) 정보 시스템 : 데이터를 저장하다 필요할때 유요한 정보를 만들어 주는 수단
	5) 데이터 베이스 : 정보시스템 안에서 데이터를 저장하다 필요할때 제공하는 역할 담당(정보시스템에 따까리)


ㄴ. 데이터베이스의 정의와 특징-정의
정의
1) 공유데이터 : 공용데이터
2) 통합 데이터 : 최소의 중복이 허용된 데이터
3) 저장 데이터 : 컴퓨터가 접근할 수있는 매체에 저장된 데이터
4) 운영 데이터 : 조직의 주요기능을 수행하기 위해 지속적으로 꼭 필요한 데이터

특징
1) 실시간 접근성
2) 계속 변화
3) 내용기반 참조
4) 동시 공유

ㄷ. 데이터와 데이터 베이스
데이터의 분류
1) 정형데이터
	: 규칙에 의해 수치만으로 쉽게 파악이 가능한 쉬운 데이터
2) 비정형 데이터
	: 규칙이 없어 의미를 쉽게 알기 힘든 데이터 ex)음성, 영상, 텍스트
3) 반정형 데이터
	: 데이터 구조와 데이터가 함께 저장되어있음 구조를 파악하는 분류과정필요(분류가 끝나면 정형데이터가된다.)
